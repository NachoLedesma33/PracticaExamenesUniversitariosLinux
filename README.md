# PracticaExamenesUniversitariosLinux

Simulador de terminal Linux para practicar comandos, con ejercicios interactivos de exámenes universitarios.

- 100% local (sin servidor, sin internet)
- Sistema de archivos virtual Linux
- 72+ comandos simulados
- ~329 ejercicios de parciales
- Validación por comando, estado del VFS y texto
- Temas claro/oscuro
- Importación de ejercicios personalizados (.md/.json)
